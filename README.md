## 🎯 Objetivos da Disciplina

Este repositório contém scripts e arquivos desenvolvidos ao longo da disciplina. O foco cobre:

- Análise de dados e tendências de mercado com ferramentas tecnológicas para tomada de decisão corporativa
- Coleta e tratamento de dados com técnicas de **Business Intelligence** para análises preditivas e descritivas
- Pesquisa de mercado: métodos qualitativos e quantitativos, etapas e aplicação
- Análise de cenários econômicos com **data mining** para suporte a decisões nos setores público e privado
- Elaboração de diagnósticos e relatórios com análise de impactos, correlações e causas
