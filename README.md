## Sobre o Repositório
Este repositório contém os scripts e arquivos desenvolvidos durante a disciplina de Inteligência de Mercado e Analytics. 
O foco é:
1. Analisar dados e tendências de mercado, utilizando técnicas e ferramentas tecnológicas
adequadas, para tomada de decisão nas corporações;
2. Coletar e tratar dados, com técnicas de business intelligence, para realizar análises
preditivas e descritivas;
3. Realizar pesquisa de mercado, considerando as técnicas disponíveis, para auxiliar na
tomada de decisão;
4. Analisar cenários econômicos, aplicando métodos de data mining, para contribuir com
tomadas de decisões nos setores públicos e privados;
5. Realizar diagnósticos e preparar relatórios, com análise de dados e metodologias
adequadas, para explicar impactos, correlações e causas em cenários econômicos

## Conteúdos da matéria
### Unidade I - Business Intelligence (BI)
● Inteligência de mercado e Business Intelligence (BI)

● Introdução ao Big Data Analytics e Inteligência Artificial (IA)

● Extração dos dados, preparação dos dados (limpeza, enriquecimento)

● Visualização e análise de dados

● Qualidade dos dados

● Análise descritiva

#### Unidade II - Contextos da Análise de Mercado
● Pesquisa de mercado: conceito; contexto; métodos qualitativos; métodos quantitativos;

etapas; aplicação; apresentação de resultados.

● Análise do mercado, concorrentes e clientes

● Gestão de relacionamento do cliente (CRM)

● Segmentação de mercado

### Unidade III - Aplicações em cenários Econômicos e Financeiros
● Ferramentas tecnológicas para inteligência de mercado

● Aplicações em cenários econômicos e financeiros

● Análise preditiva

### Unidade IV - Técnicas de Ciência de Dados
● Aplicação de sistemas como Power BI

● Métodos de Machine Learning

● Aplicação para resolução de problemas
